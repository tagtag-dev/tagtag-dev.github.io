@RestController
@Validated
@AllArgsConstructor
@RequestMapping("/users")
@Tag(name = "")
public class UserController {
    private final UserService userService;

    @PostMapping
    @OperationLog(module = "IAM", name = "", type = OperationType.ADD)
    @RequirePerm(Permissions.USER_CREATE)
    @Operation(summary = "")
    public Result<Long> create(@Valid @RequestBody UserCreateDTO dto) {
        return Result.ok(userService.create(dto));
    }
}

Core Features

Everything you need to build admin systems

Tagtag Starter provides a robust foundation for your enterprise applications, allowing you to focus on business logic.

Modular Architecture

Strictly layered design with common, framework, kernel, contract, and module. Decoupled and maintainable.

Enterprise Ready

Built-in RBAC, Operation Logs, and Rate Limiting.

Contract First

Clear separation of API definitions and implementations using DTOs and Interfaces.

Modern Stack

Built on Spring Boot 3, MyBatis Plus, and Redis, integrated with Vben Admin (Vue 3).

System Modules

Built-in Dictionary, Configuration, and Notification modules.

Developer Experience

Standardized error handling, response wrapping, and utility libraries.
